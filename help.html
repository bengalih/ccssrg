Q: What is an Access Token?

A: An Access Token is a special password setup so external tools (like this one) can access your Canvas account. Setting up an Access token takes only a minute. You can find instructions here.

back to report

Q: What is the Canvas URL?

A: The Canvas URL is the website you normally visit to access canvas. It should usually start with some form of your institution's name and end with 'instructure.com'.

back to report

Q: What is the Course List?

A: Instead of running a report against all courses, you can enter in a comma separated list of course numbers. You can find a course number for a course by looking at the URL in Canvas while viewing a course. It will look something like 'https://mysite.instructure.com/courses/#####''. The '#####' is the course number. So, to run a report agains only courses 11111 and 22222 you would enter: '11111, 22222'.

Q: What is the Download Report Only option?

A: Normally, after the report runs it will display in your browser. The 'Download Only' option will instead present the report as a file to download. If you don't choose this option you can still choose to save the report as a file from the web browser's file menu.

Q: How does this all work???

A: The CCSSRG tool works by using a set of programming interfaces (APIs) that Canvas/Instruture makes available to all Canvas users. This website uses those APIs with the Access Token you provide to pull your data from Canvas and present it in a consolidated format. Your Access token is sent straight from the report page to Canvas over a secure (https) connection. It is never stored, logged, or seen by any third party. Once your report is generated, it will be archived on the server for diagnostic purposes. The server administrator can see only the report you ran, but not run additional reports, see, or change anything else in Canvas.
