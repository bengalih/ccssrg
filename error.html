What else could be wrong?

Well, lots of things - but the ones above are the most common for you to be getting this error message.
As long as you have provided a valid Access Token and a proper Canvas URL things should work.
However...
  • This tool is intended to generate a report for STUDENT accounts. If you are running this against another type of account (Observer/parent, instructor, admin) you may get errors or inconsistent results.
  • This tool has a timeout limit. If your report runs for a long time (~3-5 minutes) and you receive this error, then you are most likely running up against this limit. The problem is because there are too many assignments in your Canvas courses (or simply too many courses!). You may be able to get around this error by running the report against only some courses using the Course List option.
